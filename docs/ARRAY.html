ARRAY function

This article describes the formula syntax of the ARRAY function.

Description

Return an array associated with handle.

Syntax

ARRAY(handle, _fast)

handle is a handle to an array of numbers.
_fast is an option boolean to specify fast lookup. Default is FALSE.

Retrieve an in-memory array created by \ARRAY. By default the handle is checked to ensure the array was created by a previous call to \ARRAY.
