\ARRAY function

This article describes the formula syntax of the \ARRAY function.

Description

Return a handle to the in-memory array.

Syntax

\ARRAY(array, _columns)

array is an array or handle to an array of numbers.
_columns is an optional number of columns. Default is `.

Create an in-memory two-dimensional array of numbers to be used by array functions. If array is a scalar and _columns is not zero then return a handle to an uninitialized array having array rows and _columns columns. Most array functions work in two modes: function and command. If the first argument is an array then an new array is returned and the array function has no side effects. If the first argument is a handle to an array then the function modifies the in-memory array and returns the array handle.

Return to index.
